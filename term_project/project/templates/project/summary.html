Foundation 6 framework


Introduction:
Foundation is an advance front-end framework created by web design company Zurb. It is a framework for designing beautiful responsive websites, apps and emails that works on all types of devices.

Popularity:

Given below is a google trend on the search interest of Foundation in the last five years:


Which developers use Foundation:
  • Developers looking for customizability at any skill level.
  • Developers looking for a framework that allows for very fast prototyping.
  • Developers who are looking for a framework who puts mobile friendliness at the forefront.

Websites that use Foundation:
  • Vans
  • Addidas
  • Arduino
  • TOMS
  • Polaroid

Strengths:
  • It has a very powerful grid system enabling the designers to push the limits of their creativity.
  • The styles are intentionally underdeveloped to allow for more creativity and flexibility by designers.
  • It uses Rem instead of pixels making it more flexible when dealing with different devices.
  • Services are offered for websites, apps and emails.
  • It comes with a large collection of styles that separates it from other front-end frameworks.
  • It offer layout templates for faster web development.
  • It offer four different types of installation to fit users development needs.
  • It offers a beautifully designed documentation with interactive tutorials.
Weaknesses:
  • Complexity makes it difficult for novice web designers.
  • Some installation file sizes are large.

Django backend framework


Introduction:

Django is an open source and free high-level python framework that makes it easier for developers to write their apps. Originally developed by Adrian Holovaty and Simon Willison, Django is maintained by the Django Software Foundation.

  • Developers that is capable of many visitors and/or transactions
Websites that use Django:
  • Pinterest
  • The Onion
  • The Washington Post
  • NASA
  • Prezi
Strengths:
  • Fast.
  • Secure
  • Exceedingly Scalable
  • Versatile
Weaknesses:
  • Too monolithic.
  • Everything is based on Django ORM.
  • Knowledge of full system is needed to work.
  • Components get deployed together.
